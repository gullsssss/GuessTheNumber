# 🎯 Guess the Number (GTN)

![Status](https://img.shields.io/badge/Status-Working-brightgreen)
![Game](https://img.shields.io/badge/Game-Terminal-orange)

A simple "Guess the Number" game written in Python that runs in the terminal.
The player tries to guess a randomly generated number within a limited number of attempts.

## 📌 Features

- 🎯 Random number generation
- 🔢 Custom range (from / to)
- ⏳ Custom number of attempts
- 📉 Hints (higher / lower)
- 🔁 Replay system after win or loss
- 🛡 Input validation (no crashes on wrong input)

## 🚀 Installation

Clone the repository:
```bash
git clone https://github.com/gullsssss/GuessTheNumber.git
```
Go to the project folder:
```bash
cd GuessTheNumber
```
Run the game:
```bash
python GTN.py
```
---

## 🎮 How to Play

1. Enter the range of numbers (e.g. 1 to 100)
2. Set how many attempts you want
3. Try to guess the number
4. Use hints:
   - "lower" → your number is too big
   - "higher" → your number is too small
5. Win or lose — and play again!

---

## 💡 Example

Enter range: 1 - 10
Attempts: 3

Guess: 5
Higher

Guess: 8
Lower

Guess: 7
🎉 You win!
